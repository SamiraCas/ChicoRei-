# Instruções para build


##  Configuração do projeto

```sh
npm install
```

### Compilar e atualizar para desenvolvimento

```sh
npm run dev
```

### Compilação para Produção

```sh
npm run build
```

### Rexecute testes de unidade com [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
